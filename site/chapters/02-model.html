Chapter 02 · pretrain / model

Build the prediction machine.

A transformer enriches each token vector by alternating communication across positions and independent computation at each position. Residual additions preserve a clean path through depth.

Tracetensor shapes through attention
UnderstandRoPE, masking, RMSNorm, and SwiGLU

1. Start with shapes, not formulas

For a batch B, sequence length T, and model width D, the main stream has shape [B, T, D]. Attention temporarily splits D into heads; the feed-forward temporarily expands it; every residual addition returns to exactly [B, T, D].

Decoder-only

This model reads only the prefix at each position and predicts what follows. “Decoder” here describes causal self-attention, not a separate encoder-decoder translation system.

Embedding

Looks up a learned D-number vector for each token ID.

Attention

Lets each token gather a weighted mixture of visible earlier token information.

Feed-forward

Applies a larger gated neural network independently to every token position.

Output logits

Projects each final vector back to one score per vocabulary token.

2. Attention is controlled communication

Queries ask what a position needs. Keys advertise what positions contain. Values carry the information. The causal mask removes future positions; RoPE makes order visible by rotating query and key coordinates according to position.

Create projections

Queries get all heads. Grouped-query attention uses fewer key/value heads, saving memory while retaining multiple query perspectives.

Split into heads

Linear projections become [batch, heads, time, head_dim], the layout expected by fast attention.

Add position and cache

RoPE encodes token order. During generation, old keys and values are reused instead of recomputed.

Mask, mix, merge

Scaled dot-product attention performs the communication, then head outputs are reshaped and projected back to model width.

3. One block: normalize, update, add

The code stays short because MLX provides the numerical building blocks. What matters is their arrangement: pre-normalization stabilizes each update and residual addition preserves the previous representation.

SwiGLU computation

One projection creates a SiLU gate, another creates content, multiplication selects useful features, and the final projection returns to D.

Transformer block

Attention and feed-forward each contribute one update through a residual +.

Complete language model

Embed IDs, repeat blocks, normalize, then reuse the embedding table as the vocabulary classifier to save parameters.

Count learned numbers

Flatten the parameter tree and sum each tensor’s element count.

4. Prove the two essential properties

The test creates a tiny model, checks output and cache shapes, then changes future tokens and verifies that earlier logits do not change. That second assertion is direct evidence that the causal mask works.

Run · fast test

Exercise a forward pass and causal cache

No dataset or trained checkpoint is needed; these are architecture tests.

python -m pytest tests/test_model.py -q
Expected: two passing tests. A first MLX run may pause briefly while Metal initializes.

Checkpoint · explain one block

Use three verbs: attention communicates, SwiGLU computes, and residual addition remembers. If those feel concrete, the training loop will be much easier.
