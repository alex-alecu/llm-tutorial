Chapter 05 · compress

Store fewer weight values.

Four-bit affine quantization groups nearby weights, represents each with one of 16 integer levels, and stores scale information that approximately reconstructs their useful range.

1. Compression is an approximation

Changing 16- or 32-bit floating values to 4-bit integers discards precision. Group scales preserve local range, so important relative differences often survive. The result is not exactly one quarter the original size because scales, metadata, tokenizer files, and some values remain.

4-bit

Four binary digits encode 2⁴ = 16 levels. “4-bit MLX” describes the numerical representation, not a universal file format. A GGUF 4-bit file belongs to a different runtime ecosystem.

3. Quantize our fine-tuned checkpoint

Always train first and quantize the finished result. This implementation intentionally rejects fine-tuning a quantized checkpoint, keeping the learning path clear.

Run · local checkpoint

Create the 4-bit copy

The original stays untouched. The command prints total size before and after.

macllm quantize --checkpoint runs/quick-story-tuned --output runs/quick-story-tuned-4bit --bits 4
Expected: a smaller directory containing weights, config, tokenizer, and quantization.json.

4. Hold generation still

Temperature 0 always chooses the highest logit, removing sampling noise. Use the same prompt and maximum length; then any output difference comes from checkpoint numerics.

Run · fair A/B

Compare original and 4-bit

Record disk size, output, warm speed, and peak memory. A tiny model may not become faster because fixed overhead dominates.

macllm generate --checkpoint runs/quick-story-tuned --prompt "Once upon" --temperature 0
macllm generate --checkpoint runs/quick-story-tuned-4bit --prompt "Once upon" --temperature 0
Interpretation: different does not automatically mean worse. Evaluate correctness and behavior across a fixed prompt set.

5. Optional: perform a 4B conversion yourself

This starts from the original Qwen weights rather than downloading a community conversion. Expect substantial disk use while the Hugging Face cache and output coexist.

Run · optional download

Convert official weights to MLX 4-bit

mlx_lm.convert is installed from MLX-LM; its library implementation is intentionally not duplicated in this small course repository.

mlx_lm.convert --model Qwen/Qwen3-4B --mlx-path runs/qwen3-4b-4bit --quantize --q-bits 4 --q-group-size 64
mlx_lm.chat --model runs/qwen3-4b-4bit

Checkpoint · say what quantization does not do

It does not add knowledge, improve training, or guarantee speed. It primarily reduces weight storage and memory, with a quality trade-off that must be measured.
