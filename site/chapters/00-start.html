Chapter 00 · orientation

Know the loop before you run it.

An LLM is a next-token predictor. Training repeats one loop: text becomes token IDs, the model predicts, loss scores the prediction, and an optimizer nudges the weights.

Understandthe five objects that move through the course
Preparea clean Python 3.11–3.13 environment
VerifyMLX, Apple silicon, and unified memory

The whole course in one sentence

Tokens go in, logits come out, loss measures the wrongness, gradients point downhill, and the optimizer changes the weights. Everything else makes that loop useful, stable, or efficient.

Token

A small text unit represented by an integer. It may be a word, part of a word, punctuation, or a byte sequence.

Tensor

A typed, multidimensional array. Batches, token IDs, weights, logits, and gradients are all tensors.

Parameter / weight

A learned number inside the model. Training changes parameters; generation reads them.

Logit

An unnormalized score for one possible next token. Higher means the model currently prefers it.

Loss

One number summarizing prediction error. Lower is better on the same data and tokenizer.

Gradient

The direction and sensitivity of loss for every weight. Backpropagation calculates it.

Optimizer

The update rule that turns gradients into small weight changes. This course uses AdamW.

Checkpoint

Saved weights plus configuration and tokenizer—the complete state needed to run the model.

Important

Pretraining, fine-tuning, and generation use the same model. Pretraining learns broad patterns from lots of text. Fine-tuning continues learning on a small, shaped dataset. Generation freezes the weights and repeatedly chooses a next token.

Set up once

A virtual environment keeps this course’s packages separate from the rest of the Mac. Editable install makes the macllm command point at the Python under src/, so code changes take effect immediately.

Run · repository root

Create and verify the environment

Run one line at a time. The downloads are Python packages; training data is downloaded later.

python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e '.[dev]'
macllm doctor
Expected: checkmarks for macOS, Apple silicon, Python, MLX packages, and at least 16 GB unified memory.
