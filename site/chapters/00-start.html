Start with a guess.

A language model reads text and guesses the next piece. We will build one, improve its guesses, and run it on this Mac.

1. The learning loop

The model repeats four steps. The technical words come second.

1 · Split

Break text into small pieces called tokens.

2 · Guess

Give tokens to the model. It scores each possible next token.

3 · Check

Compare the guess with the real next token. The error score is loss.

4 · Adjust

Change the model’s stored numbers, called weights, to reduce that error.

Three modes

Pretraining learns from much text. Fine-tuning teaches a narrower behavior. Generation uses the learned weights without changing them.

2. Set up once

A virtual environment keeps this project’s Python packages separate. The install adds the local macllm command.

Run · repository root

Create and verify the environment

Run one line at a time. The downloads are Python packages; training data is downloaded later.

python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e '.[dev]'
macllm doctor
Expected: checkmarks for macOS, Apple silicon, Python, MLX packages, and at least 16 GB unified memory.

Local
